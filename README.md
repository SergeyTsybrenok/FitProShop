# Еще одно задание

## Обновленный репозиторий
Мои поздравления! Официальный репозиторий был удален или скрыт, теперь это основной, он **никуда отсюда не денится!**
Повторяю что этот репозиторий был форком оригинальной работы переделанный под новые реали практической перечисленные ниже.

## Новые функции(требования)
- Добавление нового товара
- Добавление новых изображений (в виде ссылок)
- Изменение любого параметра товара (изображения, навания, описания, цены, гендера)
- Удаление товара

## Изменение требований
Если вдруг требования практической изменятся и этот репозиторий не будет им соответствовать, то жду ваших Pull Request'ов! Не стесняйтесь
